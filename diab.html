Project Description

The Diabetes Prediction System is a machine learningâ€“based web application designed to predict whether an individual is likely to have diabetes based on key medical and lifestyle parameters. The project focuses on supporting early diagnosis and preventive healthcare.

The backend of the application is developed using Python and Flask, where data preprocessing, feature handling, and model inference are performed. A trained classification model evaluates parameters such as glucose level, blood pressure, BMI, insulin level, age, and other medical indicators.

During development, detailed Exploratory Data Analysis (EDA) was conducted to understand feature distributions, correlations, and important predictors of diabetes. Data cleaning, handling missing values, encoding, and feature scaling techniques were applied to enhance model accuracy.

The frontend is built using HTML, CSS, and JavaScript, providing a clean, responsive, and user-friendly interface. Users can enter medical details easily through a structured input form with proper validation.

The predicted result is displayed clearly on the result page, making the output understandable even for non-technical users. This project demonstrates the practical application of machine learning in healthcare analytics.
