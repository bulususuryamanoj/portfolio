Project Description

This project focuses on identifying and predicting mental stress levels among students using a data-driven and machine learningâ€“based approach. The system aims to support early identification of stress, anxiety, and academic pressure to encourage timely awareness and intervention.

The backend is developed using Python and Flask, where data preprocessing, feature handling, and machine learning inference are implemented. The trained classification model evaluates academic workload, sleep patterns, lifestyle habits, and other factors to predict student stress levels.

Extensive Exploratory Data Analysis (EDA) was performed to identify correlations and significant indicators of stress. Data cleaning, encoding, and feature scaling techniques were applied to improve prediction reliability.

The frontend is built using HTML, CSS, and JavaScript, providing a clean, responsive, and user-friendly interface. Results are displayed in a simple and understandable format, making the application accessible to non-technical users.

This project demonstrates an end-to-end application of machine learning, data analysis, Flask development, and UI design in a real-world healthcare and education use case.
